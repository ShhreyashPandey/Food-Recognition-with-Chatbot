Problem Statement - Development of an AI-based chatbot for real-time food recognition and personalized conversation through image and text processing.

Objectives -
Achieve high accuracy in food recognition using ResNet on a custom image dataset.
Implement NLP with Flan-T5 to respond accurately to nutritional queries.
Develop an integrated user interface for seamless interaction with Chatbot.

Dataset -
Image Dataset:
Four food categories: donuts, french fries, pizza, pasta
Training: 500 images per class
Testing/Validation: 165 images per class
Preprocessing: Resizing, normalization, data augmentation (rotation, scaling, flipping)

Text Dataset:
Gathered from Wikipedia for each food category
Cleaning: Tokenization, lemmatization, stop-word removal
